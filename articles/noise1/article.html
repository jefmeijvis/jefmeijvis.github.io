Creating a sketch with P5.js

In this brief tutorial I would like to go through the steps required for creating your own P5.js sketch. If you're not familiar with P5, it's essentially a javascript library for drawing things in the browser. The original goal of P5 was to bring the easy of use and accessibility of Processing to the web.

The P5 logo

P5 has a resourceful reference page, and an online editor. If you're just getting started with programming with P5, the editor might be a good starting point. The goal of this tutorial is to bring your sketch to a standalone web page.

So let's get started. The editor we will be using is Atom, but any other coding environment will work just as well. Let's start of by creating a folder. We will name it P5project, but anything goes. Inside that folder, we'll create a new file, and call it

html.index

<html>

  <h1>Our P5 sketch</h1>

  <div class = "sketch"></div>

</html>

This will be the entry point of our page. First we'll specify that we're using html. Next up we create a H1 with our title inside. We also create a div element, this is where we will place our sketch. We will give it a class, so we can refer to it later on. Let's call it "sketch". This div will act as the container for our sketch.

We want to center the title and our sketch, so let's create a stylesheet (css). We name it

style.css

body

{

 text-align:center;

}

 

.sketch

{

 width:400px;

 height:400px;

 margin:auto;

 background-color:blue;

}

Select the body, and align everything to the center. Then create the .sketch selector, and give it a width and height of 400 pixels. Set the margin to auto, to make it align in the center, and give it a blue background color, so we can see where it is. But we can't forget to include the .css file in our .html file, otherwise it wouldn't do anything. Lets add the following line at the top of our html file, so we get this:

html.index

<html>

<link rel = "stylesheet" href="style.css">

  <h1>Our P5 sketch</h1>

  <div class = "sketch"></div>

</html>

Now we can take a look a what we have made so far, just click on the index.html file, and open it with your browser. At this point you should have a page with your title and sketch aligned in the center. The sketch is still just a 400 by 400 blue rectangle.

We're ready to continue on with the fun stuff: creating the P5 sketch. Let's create a new file, and call it

sketch.js

function setup()

{

 let canvas = createCanvas(400,400);

 let containerlist document.getElementsByClassName("sketch");

 let container = containerlist[0];

 canvas.parent(container);

 

 frameRate(60);

 textAlign(CENTER,CENTER);

 noStroke();

 fill(0);

}

 

function draw()

{

 background(255);

 text("This is our sketch",width/2,height/2);

 ellipse(mouseX,mouseY,150,150);

}

So let's have a look, what's going on here? We start of with a setup function. We create a canvas of 400 by 400 pixels, and store a reference in a local variable called canvas. We get an array containing all the elements in the document called with a classname "Sketch" , and store these in the local variable containerlist. Next up, we define our container as the first element in that list, since there is only one such element in our page. Finally we take the canvas and make the container it's parent.

We also align the text in the center, both horizontally and vertically, and set the framerate to 60 fps. We disable the stroke effect, and set our fill color to black;

In the draw function we draw a white background. We also draw some text in the middle of the sketch. Finally, we draw an circle of diameter 150 pixels at the mouse position.

To tie everything together, we add a final line to our html file, which includes our sketch.js file. Our final html.index looks as follows:

sketch.js

<html>

<link rel = "stylesheet" href="style.css">

<script src = "sketch.js"></script>

  <h1>Our P5 sketch</h1>

  <div class = "sketch"></div>

</html>

I included the sketch below, so we can have a look at our end result. From here on, the sky is our limit, as we're able to include any kind of sketch in any kind of web page. Go wild!
