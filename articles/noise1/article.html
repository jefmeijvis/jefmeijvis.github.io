Creating a sketch with P5.js

In this brief tutorial I would like to go through the steps required for creating your own P5.js sketch. If you're not familiar with P5, it's essentially a javascript library for drawing things in the browser. The original goal of P5 was to bring the easy of use and accessibility of Processing to the browser.

P5 has a resourceful reference page, and an online editor. If you're just getting started with programming with P5, the editor might be a good starting point. The goal of this tutorial is to bring your sketch to a standalone web page.

So let's get started. The editor we will be using is Atom, but any other coding environment will work just as well. Let's start of by creating a folder. We will name it P5project, but anything goes. Inside that folder, we'll create a new file, and call it

html.index

<html>

  <h1>Our P5 sketch</h1>

  <div class = "sketch"></div>

</html>

This will be the entry point of our page. First we'll specify that we're using html. Next up we create a H1 with our title inside. We also create a div element, this is where we will place our sketch. We will give it a class, so we can refer to it later on. Let's call it "sketch". This div will act as the container for our sketch.

We want to center the title and our sketch, so let's create a stylesheet (css). We name it

style.css

body

{

 text-align:center;

}

 

.sketch

{

 width:400px;

 height:400px;

 margin:auto;

 background-color:blue;

}

Select the body, and align everything to the center. Then create the .sketch selector, and give it a width and height of 400 pixels. Set the margin to auto, to make it align in the center, and give it a blue background color, so we can see where it is. But we can't forget to include the .css file in our .html file, otherwise it wouldn't do anything. Lets add the following line at the top of our html file, so we get this:

html.index

<html>

<link rel = "stylesheet" href="style.css">

  <h1>Our P5 sketch</h1>

  <div class = "sketch"></div>

</html>
